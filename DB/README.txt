La Base de Datos fue creada en mysql y cotejamiento latino.

La estructura de la Base de Datos se encuentra en la carpeta Estructura,
en el cual esta un archivo .sql el cual puede ser importado facilmente
con phpmyadmin paro lo cual primero deberan crear una Base de Datos de 
nombre: RegisterLU., uno .tex que para ser generado un pdf u otro formato
con el y por ultimo un .xml.


Las Tablas estan diseñadas con el motor de almacenamiento Innodb.

Documentacion acerca de la Base de Datos en la carpeta doc de este
nivel con archivos .pdf
